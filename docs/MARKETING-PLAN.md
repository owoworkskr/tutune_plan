## 1. 세대 타겟 결론부터

### 답: 둘 중 하나 고르는 문제가 아니다. **Z로 퍼뜨리고, 밀레니얼로 수금한다.**

| | Z세대 (10대 후반~20대 중반) | 밀레니얼 (20대 후반~30대) |
|---|---|---|
| 역할 | **확산 엔진** — 공유·밈·유형놀이 | **지갑** — 심층 콘텐츠 결제 |
| 소비 문법 | "나는 이 그룹에 속해" (유형 소속으로 자기 증명) | "내 고민에 답을 달라" (이직·결혼·육아·번아웃) |
| 맞는 상품 | 갑자 카드, 최애 궁합, 스낵 콘텐츠 (무료+광고) | 심층 카드, 대운 가이드 $10, 연령대 패키지 |
| 맞는 채널 | 틱톡, 릴스, X | 인스타 피드, 검색, 커뮤니티 |
| 결제 기대 | 낮음 (광고 수익 + 팁 소액) | 상대적으로 높음 (고민 해소에 지불) |

### 근거 (확인된 데이터)
- 미국 설문 기준 Z세대와 젊은 밀레니얼의 80%가 점성술을 믿는다고 답했고, 틱톡 astrology 해시태그 영상은 450만 개 규모
- Co-Star는 2020년 750만 → 2023년 3,000만 유저로 성장했는데, 마케팅 비용을 쓰지 않고 이 성장을 만들었다고 밝힘 — **공유 설계가 광고를 이긴 사례**
- 점성술 관련 지출은 2021년 128억 달러에서 2031년 228억 달러 규모로 성장 전망
- Pew 조사에서 밀레니얼의 약 60%가 뉴에이지형 영성을 자기 신앙으로 여기고 약 30%가 점성술을 믿는 것으로 나타남 — **밀레니얼도 시장에서 빠진 적이 없음**

### 투튠 구조에 대입하면
이미 답이 제품에 들어있음. **연령대 패키지(20대=Z 취업·연애·덕질 / 30대=M 이직·결혼·육아)** 가 세대 이원화 그 자체. 마케팅만 채널별로 크리에이티브를 갈라주면 됨.

```
Z 퍼널:  틱톡/릴스 갑자 카드 밈 → 웹 랜딩 카드 뽑기 → 앱 설치 → 광고 수익
M 퍼널:  인스타/커뮤 "왜 난 안 풀리는가" 훅 → 앱 설치 → 심층 결제 → 매출
```

리텐션 명제(0.5~2%가 매출 50%)와 정합: 그 0.5~2%는 대부분 M 퍼널에서 나온다.

## 4. 콘텐츠 제작 자동화 실행안

### 이미 있는 것 (재사용)
- Make.com: 뉴스 수집→Claude→Discord 봇, 월간·주간 콘텐츠 파일 생성기
- Figma 자동 채움: `fill_fortune.py` — 12지신 프레임 자동 생성 검증됨
- Claude API

### 이미지 (인스타 피드) — 지금 바로 가능, ₩0
```
Make 스케줄 → Claude API (띠별운세 텍스트 생성)
→ fill_fortune.py 방식으로 Figma 템플릿 채움 → 이미지 익스포트
→ Meta Graph API (Instagram Content Publishing)로 자동 게시
```
- 필요한 것: Meta Developer 앱 (진행 중), 인스타 비즈니스 계정 + 페이스북 페이지 연결
- Make 무료 플랜 시나리오 2개 제한 걸리면: 발행 시나리오를 GitHub Actions로 이관 가능 (아래 방식과 동일)

### 쇼츠 영상 (틱톡/릴스/유튜브) — ₩0 파이프라인
```
운세 텍스트 (Claude) → TTS 음성 (Edge-TTS, 무료) 
→ ffmpeg: 카드 이미지 + 자막 + 음성 + BGM 합성 → 15~30초 세로 영상
```
- 전부 오픈소스. GitHub Actions나 로컬 파이썬으로 주 단위 배치 생성 가능
- 유료 대안 (편하지만): Creatomate, JSON2Video — 월 $40~ 수준이라 지금 단계엔 비추

### 게시 자동화 — 채널별 현실
| 채널 | 자동 게시 | 방법 | 솔직한 상태 |
|---|---|---|---|
| Instagram | ⭕ 가능 | Meta Graph API (릴스 포함) | 무료, 이미 진행 중인 연동으로 됨 |
| YouTube Shorts | ⭕ 가능 | YouTube Data API v3 | 무료. OAuth 세팅 한 번이 귀찮은 정도 |
| TikTok | 🔺 제한적 | Content Posting API는 앱 심사 필요 | **1인 개발엔 장벽 높음. 영상 생성만 자동화하고 업로드는 수동(주 2회 몰아서)이 현실적** |
| X | ⭕ 가능 | X API 무료 티어 (월 500포스트) | 웹 랜딩 링크 뿌리기엔 충분 |

결론: **"생성은 전자동, 게시는 인스타·유튜브·X 자동 + 틱톡만 수동"** — 이게 지금 리소스에서 지속 가능한 최대치.

## 5. 주간 동향 자동 업데이트 시스템

### 구조 (GitHub Actions — Make 시나리오 안 잡아먹음, 무료)
```
매주 월요일 09:00 KST (GitHub Actions cron)
→ 수집: Google News RSS (운세앱·astrology app·K-fortune 등 키워드)
        + Apple 앱 랭킹 JSON (KR/TH/BR 라이프스타일 무료 Top — 경쟁앱 순위 추적)
        + Reddit r/astrology 주간 인기글 RSS
→ Claude API: 요약 + "투튠에 의미 있는 것" 3줄 도출
→ 커밋: docs/trends/2026-Wxx.md (아카이브) + dashboard/trends.json (대시보드 표시용)
```

### 셋업 (한 번만, 5분)
1. 레포에 이번 zip의 `.github/workflows/weekly-trends.yml` + `scripts/collect_trends.py` 커밋
2. 레포 Settings → Secrets and variables → Actions → `ANTHROPIC_API_KEY` 등록
3. Actions 탭에서 "Weekly Trends" 수동 Run 한 번 돌려서 확인
4. 끝. 이후 매주 자동. 대시보드 열면 최신 동향이 상단에 뜸

### 비용
- GitHub Actions: 무료 한도 내 (주 1회 몇 분)
- Claude API: 주 1회 호출, 월 수백 원 수준
- API 키 필요한 건 Anthropic 하나뿐. 나머지 소스는 전부 키 없는 공개 피드

### 확장 여지 (필요해지면)
- Naver DataLab API 추가 (키 이미 보유) → KR 검색 트렌드 병합
- Discord 웹훅 추가 → 대시보드 + 디스코드 동시 수신 (기존 CMO 봇과 합류)
